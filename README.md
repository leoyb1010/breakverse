把今天烦人的小事变成圆滚滚怪兽，选装备，开打，掉落新玩具，再生成一张适合分享的搞笑战报。

## 快速开始

离线单机模式：

```bash
# 直接打开
open index.html
```

账号、云存档和管理员面板模式：

```bash
python3 server.py
```

默认访问 `http://localhost:4173`。核心游戏仍然是一个自包含的 `index.html`，`server.py` 只负责账号、会话和云存档 API。

> 当前仓库未添加正式开源许可证，默认仅供个人学习、试玩和内部预览使用；商用或二次发布请先获得授权。

## 当前玩法

- 输入一句今日小烦事，本地生成“今日最烦海报”怪兽，也可以直接点预设开局。
- 选择大关卡、难度、开局物种，再搭配四个装备槽：主武器、副武器、技能一、技能二。
- 进入画布横版动作战斗：移动、跳跃、主武器三连段、完美闪避、副武器、主动技能、怒气大招。
- 每个大关卡都有 5 个连续小关路线：热身、精英、机制、收集、最终烦人兽；战斗中会显示当前进度带。
- 怪兽有多阶段、清晰预警、弱点发光、主题环境互动。
- 小怪、精英和小关结算会按关卡属性掉落对应装备；大关卡、怪兽、成就、彩蛋目标会写入本地 `localStorage`。
- “我的爆破宇宙”已经拆分为装备库、大关卡、成就、怪兽图鉴、彩蛋目标，避免所有系统混在一起。
- 每日目标会根据日期生成，引导玩家刷不同大关卡、连击、分享和彩蛋。
- 结算页会根据难度、成就、连击和掉落生成更适合晒出去的接力文案，也可下载图片战报。

## 难度

| 难度 | 变化 |
| --- | --- |
| 卷 | 轻松爆破，怪物血量和伤害较低 |
| 很卷 | 标准挑战，掉落略好 |
| 非常卷 | 怪物更快更硬，掉落更香 |
| 卷死 | 高压挑战，最高掉落倍率和专属成就 |

## 操作

| 按键 | 动作 |
| --- | --- |
| `A` / `D` 或 `←` / `→` | 左右移动 |
| `W` / `空格` / `↑` | 跳跃 |
| `J` | 轻攻击三连段 |
| `K` | 闪避，时机好会触发完美闪避 |
| `L` | 副武器；怒气满时释放大招 |
| `U` | 技能一 |
| `I` | 技能二 |

移动端进入战斗后会切换为固定视口：左侧为移动/跳跃，右侧为攻击/闪避/技能，页面不会在战斗中滚动。

## 账号系统

当前服务端已内置账号 API，无需额外依赖：

- 注册、登录、退出登录。
- 密码使用 PBKDF2-HMAC-SHA256 哈希保存。
- 会话使用 `HttpOnly` Cookie，默认 7 天有效期。
- 首个注册账号自动成为管理员。
- 登录后可上传当前存档、拉取云存档。
- 管理员可查看账号列表、启用/停用账号、调整玩家/管理员角色。

账号数据默认写入：

```bash
data/accounts.json
```

该文件已加入 `.gitignore`，不要提交到仓库。正式部署时可用环境变量调整：

```bash
PORT=4173
BREAKVERSE_DATA_DIR=/path/to/private-data
BREAKVERSE_SESSION_TTL=604800
BREAKVERSE_COOKIE_SECURE=1
```

## 技术结构

`index.html` 内部包含：

- 数据驱动内容表：主题、首领怪、装备、掉落。
- 大关卡系统：近战、远程、远近混合、躲避跑酷四条路线，每条路线包含 5 个小关配置。
- 难度系统：卷、很卷、非常卷、卷死四档影响血量、伤害、速度、攻击频率和掉率。
- 开局系统：物种属性、快速预设和关卡推荐负载。
- 装备负载：主武器、副武器、双主动技能、属性与趣味被动。
- 本地生成器：关键词匹配、随机变体、公开文本改写。
- 存档系统：装备、图鉴、大关卡通关、成就、统计、历史战报、今日目标。
- 战斗系统：物理、碰撞、三连段、完美闪避、主动技能、怒气、怪兽招式、路线机制、小关结算、随机关卡彩蛋。
- 渲染系统：画布视差背景、角色装备层叠、粒子、伤害数字、战斗提示。
- 分享系统：复制文案、画布图片战报。
