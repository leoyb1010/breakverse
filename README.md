# 今日烦人兽爆破局

把今天烦人的小事变成圆滚滚怪兽，选装备，开打，掉落新玩具，再生成一张适合分享的搞笑战报。

## 快速开始

```bash
# 直接打开
open index.html

# 或启动本地服务
python3 -m http.server 4173
# 浏览器打开 http://localhost:4173
```

无需安装、无需构建、无外部依赖。核心游戏仍然是一个自包含的 `index.html`。

## 当前玩法

- 输入一句今日小烦事，本地生成“今日最烦海报”怪兽。
- 选择四个装备槽：主武器、副武器、技能一、技能二。
- 进入 Canvas 横版动作战斗：移动、跳跃、主武器三连段、完美闪避、副武器、主动技能、怒气大招。
- 每局是 5 站路线：小怪热身、精英岔路、技能试炼、装备小卖部、最终烦人兽。
- 怪兽有多阶段、清晰预警、弱点发光、主题环境互动。
- 小怪和精英有概率掉落装备或碎片，胜利后最终掉落也会存入本地 `localStorage`。
- 结算页可复制分享文案，也可下载图片战报。

## 操作

| 按键 | 动作 |
| --- | --- |
| `A` / `D` 或 `←` / `→` | 左右移动 |
| `W` / `空格` / `↑` | 跳跃 |
| `J` | 轻攻击三连段 |
| `K` | 闪避，时机好会触发完美闪避 |
| `L` | 副武器；怒气满时释放大招 |
| `U` | 技能一 |
| `I` | 技能二 |

移动端会显示触控按钮。

## 技术结构

`index.html` 内部包含：

- 数据驱动内容表：主题、Boss、装备、掉落。
- 装备负载：主武器、副武器、双主动技能、属性与趣味被动。
- 本地生成器：关键词匹配、随机变体、公开文本改写。
- 存档系统：装备、图鉴、统计、历史战报。
- 战斗系统：物理、碰撞、三连段、完美闪避、主动技能、怒气、怪兽招式、环境互动。
- 渲染系统：Canvas 视差背景、角色装备层叠、粒子、伤害数字、战斗提示。
- 分享系统：复制文案、Canvas 图片战报。
